# Amber Kim: React Redux Excercise in fetching data from a server

## Description
A Budget and expense tracking app.

## Technologies
* React
* Redux
* Node
* Mongoose

## Instructions
Run the front end separately from the back end.

## Running the back end
Navigate to the 'backend' folder in 'lab-amber'.

Make a copy of .env.sample and rename it to .env.

Run this command in your terminal:

```
node server.js
```

## Running the front end
Navigate to the 'frontend' folder in 'lab-amber' then run this command in your terminal: 

```
npm run hot
```
